# **App Name**: GloAds Portal

## Core Features:

- Dashboard Overview: Displays the current wallet balance and a prominent 'Deposit Money' button.
- Ad Campaign Form: Form to input ad details including Ad Title, Target Link, Video Upload, and Placement selection.
- Video Upload to Cloudinary: Allows users to directly upload ad videos to Cloudinary.
- Placement Selection: Allows users to select ad placement from predefined options (Home Top, In-Shorts Feed, Search Top, Video Start).
- Reach Estimator: Budget slider that estimates audience reach based on the input amount (â‚¹80 = 700-900 views).
- Ad Campaign Submission: Submits the ad campaign to Firestore with 'Pending' status under the 'ad_campaigns' collection in the gloverse-d94dc database.

## Style Guidelines:

- Primary color: Gold (#FFD700) for a premium and luxurious feel.
- Background color: Very dark gray (#0F0F0F) to create a dark UI theme.
- Accent color: Slightly lighter gray (#333333) for contrast and depth.
- Headline font: 'Poppins', a geometric sans-serif font, offering a precise and contemporary feel suitable for short titles and prominent text.
- Body font: 'Inter', a grotesque-style sans-serif providing a modern, objective, neutral look that ensures comfortable readability in the body.
- Use minimalist icons with gold accents to maintain the premium feel.
- Dark themed layout, dividing information in cards and sections for easy understanding.
